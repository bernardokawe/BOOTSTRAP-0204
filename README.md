#  **🍕 SAPORE - Cozinha Italiana**

## *“Sapore: onde cada prato transforma ingredientes em memórias inesquecíveis.”*

Trazemos a tradição italiana com receitas autênticas totalmente artesanais e feitas com paixão garantindo o sabor único que só a Sapore tem.

## 🤌 Sobre o Projeto
Este é um site simples de uma única página criado para um exercício de front-end usando o framework Bootstrap.

### 🍝 Funcionalidades
- Layout responsivo (funciona em mobile e desktop).
- Seções: Home, Menu, Sobre, Localização, Contato.
- CSS com Bootstrap.
